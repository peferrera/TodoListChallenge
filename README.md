# To-Do List Challenge
A to-do list application.

## Requirements

- Node.js (recommended: 4.x)
- npm (recommended: 2.x or 3.x)

## Setup

```sh
npm install
```

Para criar o build da aplicação

```sh
npm run build
```

e para continuar recriando o build enquanto trabalha, use:

```sh
npm start
```
